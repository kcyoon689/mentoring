결정 트리의 한계점과 앙상블의 필요성

결정 트리의 한계점

과적합 경향

트리가 복잡해질수록 훈련 데이터에만 과도하게 최적화되어 새로운 데이터에 대한 예측 성능이 저하됩니다. 가지치기와 같은 규제 기법이 필수적입니다.

해결책: 사전/사후 가지치기 적용

높은 분산

훈련 데이터의 작은 변화에도 트리의 구조가 크게 달라질 수 있어 모델의 안정성이 떨어집니다. 이는 예측 결과의 일관성을 저하시킵니다.

해결책: 여러 트리를 결합한 앙상블

탐욕적 알고리즘

결정 트리는 각 단계에서 최적의 분할을 찾는 탐욕적인 방식으로 작동하여, 전체적으로 최적인 트리를 보장하지 못할 수 있습니다.

해결책: 다양한 관점의 모델 조합

계단 형태의 결정 경계: 축에 평행한 결정 경계만을 생성하므로, 복잡한 데이터 분포에 대한 예측 성능이 저하될 수 있습니다.

앙상블 기법의 필요성

앙상블 기법은 여러 개의 학습 모델을 결합하여 단일 모델의 한계를 극복하고 예측 성능을 향상시킵니다.

분산 감소

여러 모델의 예측을 결합함으로써 개별 모델의 오류가 상쇄되어 전체적인 분산이 감소합니다.

과적합 방지

여러 모델을 사용함으로써 훈련 데이터의 노이즈에 과도하게 적응하는 것을 방지합니다.

예측 정확도 향상

다양한 관점에서 학습된 모델들의 '집단 지성'을 활용하여 더 정확하고 안정적인 예측이 가능합니다.

복잡한 결정 경계

여러 트리의 조합으로 더 유연하고 복잡한 결정 경계를 형성하여 비선형 관계도 잘 포착할 수 있습니다.
