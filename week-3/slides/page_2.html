결정 트리의 기본 개념

정의

결정 트리는 분류와 회귀 문제 모두에 활용되는 강력하고 대중적인 지도 학습 알고리즘입니다. 이 모델의 가장 큰 장점은 모델의 의사결정 과정을 시각적으로 이해하기 쉬워 '화이트박스(White-box)' 모델로 불린다는 점입니다.

구조

  • 루트 노드 (Root Node): 트리의 가장 상위에 위치하는 시작점으로, 전체 데이터를 포함합니다.
  • 내부 노드 (Internal Node / Decision Node): 데이터를 분할하는 기준이 되는 노드로, 특정 속성에 대한 질문(테스트)을 나타냅니다.
  • 가지 (Branch): 내부 노드의 질문에 대한 답변으로, 다음 노드로 연결되는 경로입니다.
  • 리프 노드 (Leaf Node / Terminal Node): 트리의 가장 마지막에 위치하는 노드로, 더 이상 분할되지 않으며 최종적인 예측 값을 나타냅니다.

작동 원리

결정 트리는 '분할 정복(divide and conquer)' 전략을 사용하여 데이터를 반복적으로 분할합니다.

  1. 최적의 분할 기준 탐색: 불순도(Impurity) 또는 정보 획득(Information Gain)을 사용하여 분할의 품질을 평가
  2. 데이터 분할: 선택된 기준에 따라 데이터를 두 개 이상의 하위 집합으로 나눔
  3. 재귀적 반복: 나누어진 각 하위 집합에 대해 1, 2번 과정을 반복
  4. 중단: 데이터가 모두 동일한 클래스에 속하거나, 설정된 최대 깊이에 도달하거나, 노드의 데이터 개수가 적은 경우

예시

'테니스를 칠 것인가?'를 예측하는 결정 트리는 다음과 같은 규칙을 가질 수 있습니다. 먼저 '날씨'를 기준으로 데이터를 분할하고, '맑음'인 데이터에 대해서는 '습도'를 기준으로 다시 분할하는 방식입니다. 이 과정을 통해 최종적으로 '테니스를 친다' 또는 '치지 않는다'는 결론에 도달하게 됩니다.
