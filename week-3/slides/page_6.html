에이다부스트(AdaBoost)

순차적 학습 방식

에이다부스트는 여러 개의 약한 학습기를 순차적으로 학습시켜 이전 모델의 오류를 보완해나가는 직렬 방식의 부스팅 알고리즘입니다. 모델을 한 번에 하나씩 순차적으로 구축하며, 잘못 분류된 데이터에 가중치를 부여하여 다음 모델이 해당 데이터를 더 집중적으로 학습하도록 만듭니다.

작동 원리

  1. 초기 가중치 할당: 모든 학습 데이터 샘플에 동일한 가중치(w = 1/N)를 부여
  2. 약한 학습기 학습: 현재 데이터 가중치를 반영하여 약한 학습기(예: 결정 스텀프)를 학습시킴
  3. 학습기 성능 계산:
    • 총 오류(Total Error, TE): 잘못 분류된 샘플들의 가중치 합계
    • 학습기 가중치(α): α = 0.5 ln((1-TE)/TE)
  4. 데이터 가중치 갱신:
    • 정답을 맞힌 샘플: 가중치 ↓ (w ← w·e-α)
    • 틀린 샘플: 가중치 ↑ (w ← w·eα)
  5. 반복: 2~4단계를 반복하여 여러 개의 약한 학습기를 생성
  6. 최종 예측: 가중 투표(weighted majority vote)로 최종 결정

장점

  • 단일 모델보다 높은 예측 정확도
  • 다양한 종류의 약한 학습기와 결합 가능
  • 과적합에 대한 저항성이 비교적 강함
  • 구현이 비교적 단순하고 하이퍼파라미터 튜닝이 덜 복잡

단점

  • 노이즈 및 이상치에 민감
  • 순차적 학습으로 병렬 처리 어려움
  • 복잡한 학습기 사용 시 과적합 위험
