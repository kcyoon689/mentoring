지니 불순도와 속성 중요도

지니 불순도 (Gini Impurity)

지니 불순도는 결정 트리에서 특정 노드의 데이터가 얼마나 다양한 클래스로 구성되어 있는지를 측정하는 지표입니다. 즉, 노드의 '순수도'를 나타내며, 값이 낮을수록 노드가 순수하다는 의미입니다.

Gini = 1 - Σ(pj2)

  • 지니 불순도 = 0: 노드 내의 모든 데이터가 단 하나의 클래스에 속하는 경우로, 가장 순수한 상태입니다.
  • 지니 불순도 = 0.5 (이진 분류의 경우): 데이터가 모든 클래스에 걸쳐 균등하게 분포된 경우로, 가장 불순한 상태입니다.

속성 중요도 (Feature Importance)

속성 중요도는 모델이 예측을 수행할 때 각 입력 변수(속성)가 얼마나 중요한 역할을 했는지를 나타내는 점수입니다. 결정 트리에서는 특정 속성이 트리의 노드를 분할할 때 불순도를 얼마나 감소시켰는지를 기준으로 중요도를 계산합니다.

지니 중요도(Gini Importance) 또는 불순도 감소(Mean Decrease in Impurity)

각 속성이 사용된 모든 분할에서 발생한 불순도 감소량의 평균을 계산하여 중요도를 측정합니다. 어떤 속성이 트리의 상위 노드에서 자주 사용되고 불순도를 크게 감소시켰다면, 그 속성의 중요도는 높게 평가됩니다.

지니 불순도 예시

한 노드에 'Yes' 클래스 4개와 'No' 클래스 6개가 있다면:

  • pYes = 4/10 = 0.4
  • pNo = 6/10 = 0.6
  • Gini = 1 - (0.4² + 0.6²)
  • = 1 - (0.16 + 0.36)
  • = 1 - 0.52 = 0.48
