서론: 자연어 사전학습모델의 개요

자연어 사전학습 모델이란?
  • 대규모 텍스트 데이터를 미리 학습 — 언어의 복잡한 패턴, 문법, 의미, 맥락을 내재화하는 AI 모델

  • LLM 시대의 시작 — BERT·GPT는 ChatGPT, Gemini 등 현재 LLM 시대의 근간

BERT 사전학습 ① — 마스크 언어 모델 (MLM)

BERT 학습의 핵심. 문장의 일부 토큰을 숨긴 후 좌·우 전체 문맥으로 예측 → 진정한 양방향 언어 표현 학습.

MLM 동작 메커니즘
1
15% 토큰 무작위 선택
학습 문장에서 전체 토큰의 약 15%를 무작위로 선택합니다.
2
세 가지 방식으로 처리
  • 80% → [MASK] 토큰으로 대체 (모델이 예측하도록)
  • 10% → 임의 다른 단어로 대체 (노이즈 주입)
  • 10% → 원본 그대로 유지 (모델이 교란에 민감하지 않도록)
3
양방향 문맥으로 예측
마스킹된 위치를 예측할 때 해당 위치의 좌·우 문맥 모두 활용. 이것이 단방향 LM과의 근본적인 차이점입니다.
손실 함수: 마스킹된 토큰에 대해서만 Cross-Entropy Loss 계산. 나머지 토큰은 손실에 기여하지 않습니다.

기존 언어 모델과의 방향성 차이
기존 LM → 단방향 P(w_t | w₁…w_{t-1}) — 이전 토큰만 참조

GPT 시리즈의 발전 과정 — 규모 확장과 학습 전략 변화

Scaling Law: 단순한 다음 단어 예측(Next Token Prediction)을 초대규모 데이터로 반복하면 언어 이해·추론·생성 능력이 창발(Emergence)한다. 파라미터 수·데이터량·연산량을 함께 늘릴수록 성능이 예측 가능하게 향상 (Kaplan et al., 2020)

주요 응용 분야 비교 & 결론

BERT 응용 (NLU)
  • 텍스트 분류 — 감정분석·스팸 탐지·토픽 분류
  • 개체명 인식 (NER) — 인명·지명·기관명 식별
  • 질의응답 — SQuAD, KorQuAD 등 독해 QA
  • 문장 유사도·관계 분류
  • Google 검색 랭킹 (2019년~)
  • 의료·법률 특화 모델 (BioBERT, LegalBERT 등)
GPT 응용 (NLG)
  • 대화형 AI 챗봇 — ChatGPT, Claude 등
  • 코드 생성 — GitHub Copilot (Codex 기반)
  • 창의적 글쓰기·기사·광고 카피 작성
  • 텍스트 요약·기계 번역
  • 멀티모달 생성 — GPT-4V (이미지 이해 + 텍스트)
  • In-context Learning / Few-shot 추론

핵심 요약

  • BERT는 양방향 MLM 학습으로 깊은 문맥 이해 획득 → 분류·추출·이해 태스크에 최적
  • GPT는 단방향 자기회귀 학습으로 자연스러운 텍스트 생성 능력 확보 → 대화·창작·생성 태스크에 최적
  • 두 모델은 학습 방향(양방향 vs 단방향)과 목적(이해 vs 생성)이 다르며, 서로 상호 보완적인 NLP의 두 축
