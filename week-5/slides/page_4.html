AutoEncoder란 무엇인가?

4/13

특징

  • Unsupervised Learning: 라벨 없이 학습
  • 입력 = 출력: 자기 자신을 복원
  • Bottleneck Structure: 정보 압축

학습 목표

Loss = ||x - x̂||² 를 최소화

x: 입력, x̂: 복원 출력, z: 압축된 잠재표현

장점

  • Encoder: 최소한 학습 데이터는 잘 복원 (성능 보장)
  • Decoder: 최소한 학습 데이터 분포의 샘플 생성 가능

구조

Input → Encoder → Latent Space (z) → Decoder → Output

Latent Variable = Feature = Hidden Representation

차원 축소

핵심 특성 추출 및 압축
