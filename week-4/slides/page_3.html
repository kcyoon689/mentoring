다층 퍼셉트론(MLP)

구조적 특징

  • 은닉층: 입력층과 출력층 사이에 하나 이상의 은닉층을 추가
  • 완전 연결: 각 층의 뉴런들은 이전 층의 모든 뉴런과 연결
  • 비선형 활성화: 각 층에서 비선형 활성화 함수 적용

등장 배경

  • 단층 퍼셉트론은 XOR 문제와 같이 비선형적으로 분리 불가능한 문제에 한계
  • 마빈 민스키의 증명으로 인한 인공신경망 연구의 "침체기" 경험
  • 비선형 활성화 함수의 도입으로 이러한 한계를 극복

MLP의 장점

비선형 문제 해결 가능

XOR 문제와 같이 입력값이 다를 때만 1을 출력하고 같을 때는 0을 출력하는 비선형 관계 학습 가능

복잡한 패턴 학습 가능

은닉층의 여러 층을 통해 데이터를 여러 번 비선형적으로 변환하여 복잡한 패턴 인식

함수의 합성 가능

여러 비선형 함수의 합성으로 매우 복잡한 함수 표현 가능
