AI 분야의 주요 직무

AI 분야의 직무는 크게 연구 중심, 서비스 구현 중심, 및 배포 운영 중심으로 나눌 수 있습니다. 각 직무는 요구하는 역량과 수행하는 역할에서 차이를 보입니다.

AI/ML Researcher

주요 역할

  • 최신 기술 동향 파악
  • 새로운 솔루션 연구 및 개발
  • 논문 작성

요구 역량

  • 연구 능력
  • 논문 경험
  • 깊이 있는 ML 지식

AI/ML Engineer

주요 역할

  • 검증된 ML 방법론을 서비스에 구현 및 파인튜닝
  • API 개발

요구 역량

  • ML 지식
  • API 구성(FastAPI)
  • 배포(Kubernetes)
  • 백엔드 개발 지식

MLOps Engineer

주요 역할

  • 데이터 수집부터 학습, 배포 전반의 파이프라인 관리

요구 역량

  • 머신러닝 지식
  • SW/클라우드 전문가 역량
  • Docker, Kubernetes, mlflow, BentoML 등

다른 특화된 직무

ML Modeler: 머신러닝 모델 개발 전문가

Prompt Engineer: 프롬프트 최적화 전문가

Data Labeler: 데이터 라벨링 전문가

NLP/추천/컴비전: 세부 전문 분야
