결측치 구조 분석

호텔 예약 데이터에서 결측치 패턴을 시각화하고, 결측 여부가 취소율에 미치는 영향을 정리합니다.

1) 상관계수 개념

상관계수(correlation coefficient) r는 두 변수 간의 선형적 관계(직선적인 관계)의 강도와 방향을 나타내는 값입니다.

  • 범위: -1 ≤ r ≤ 1
  • r = 1 : 완벽한 양의 상관관계 → 공부 시간 ↑ → 시험 점수 ↑
  • r = -1 : 완벽한 음의 상관관계 → TV 시청 시간 ↑ → 시험 점수 ↓
  • r = 0 : 상관 없음 (선형 관계 거의 없음)

∣r∣ 값이 클수록 (즉, -1이나 1에 가까울수록) 관계가 강합니다.

2) 결측치 상관 구조

  • company ↔ agent
    • 상관 ≈ -0.59 한쪽이 결측일 때 다른 쪽은 채워진 경우가 많음
    • company 대부분 결측 → 개인 고객 중심
    • agent 결측 적음 → 중개인 예약 다수
    • 즉, 법인/단체 계약 없음 ≈ 중개인 없음과 유사한 패턴
  • country, children
    • 다른 변수와의 상관이 거의 없음 → 결측이 독립적으로 발생

3) 결측 여부와 취소율

  1. 회사 정보(company_missing)
    Not Missing 17.5% → Missing 38.2%
    회사 계약 고객 < 개인 고객의 취소율
  2. 중개인 정보(agent_missing)
    Not Missing 39.0% → Missing 24.7%
    OTA/중개인 예약이 직접 예약보다 취소율 높음
  3. 국가 정보(country_missing)
    Not Missing 37.1% → Missing 13.7%
    결측은 극소수이며 취소율 낮음 → “Unknown” 범주 처리
  4. 아동 수(children_missing)
    Not Missing 37.0% → Missing 100%
    표본 4건으로 의미 제한적 → 0으로 대체 권장
