선형 회귀와 다중 선형 회귀

선형 회귀는 통계학에서 파생된 지도 학습 기법으로, 연속적인 값을 예측하는 데 사용됩니다. 하나 이상의 독립 변수와 종속 변수 사이의 선형 관계를 모델링합니다.

단순 선형 회귀

하나의 독립 변수와 하나의 종속 변수 간의 관계를 직선으로 표현

Y = aX + b
  • Y: 종속 변수 (예측하려는 값)
  • X: 독립 변수 (예측에 사용되는 값)
  • a: 기울기 (독립 변수가 한 단위 증가할 때 종속 변수의 변화량)
  • b: y절편 (독립 변수가 0일 때의 종속 변수 값)

단순 선형 회귀 시각화

다중 선형 회귀

두 개 이상의 독립 변수를 사용하여 종속 변수를 예측하는 모델

Y = b₀ + b₁X₁ + b₂X₂ + ... + bₙXₙ
  • Y: 종속 변수
  • X₁, X₂, ..., Xₙ: 독립 변수들
  • b₀: 절편
  • b₁, b₂, ..., bₙ: 각 독립 변수에 대한 회귀 계수
