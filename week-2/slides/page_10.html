정규화와 하이퍼파라미터 최적화

정규화는 모델의 복잡도를 제어하여 과적합을 방지하는 대표적인 기법입니다. 손실 함수에 모델 가중치의 크기에 대한 페널티 항을 추가하여, 가중치 값이 너무 커지는 것을 억제합니다.

정규화 기법

L1 정규화 (Lasso 회귀)

페널티 항으로 가중치의 절댓값 합(L1 Norm)을 사용

특성 선택을 자동으로 수행

중요하지 않은 특성을 완전히 제외

가중치를 0으로 만듦

L2 정규화 (Ridge 회귀)

페널티 항으로 가중치의 제곱 합(L2 Norm)을 사용

가중치를 0에 가깝게 만듦

특성 간 상관관계가 높을 때도 안정적인 성능

모든 특성을 유지하면서 영향력을 줄임

하이퍼파라미터 최적화

하이퍼파라미터(Hyperparameter)

모델이 데이터로부터 학습하는 파라미터(가중치 등)와 달리, 사용자가 모델 학습 전에 직접 설정해야 하는 값
