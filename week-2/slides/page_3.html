행렬 연산의 이해

머신러닝에서 데이터는 주로 행렬과 벡터 형태로 표현됩니다. 행렬의 구조를 변형하거나 특정 연산을 수행하기 위해 전치 행렬과 단위 행렬의 개념이 활용됩니다.

전치 행렬 (Transpose Matrix)

기존 행렬의 행과 열을 서로 맞바꾼 행렬을 의미합니다. 원본 행렬 A에 대해 AT로 표기합니다.

A =
1 2 3
4 5 6
AT =
1 4
2 5
3 6

성질

  • 크기: m × n 행렬을 전치하면 n × m 크기의 행렬이 됩니다.
  • 성질: (AT)T = A
  • 성질: (A + B)T = AT + BT
  • 성질: (AB)T = BTAT

머신러닝에서의 응용

데이터의 구조를 변환하거나 특정 계산(예: 공분산 행렬 계산)을 용이하게 할 때 사용됩니다. 특히, 입력 데이터 X와 출력 데이터 Y의 관계를 표현할 때 유용합니다.

단위 행렬 (Identity Matrix)

주대각선의 원소는 모두 1이고 나머지 원소는 모두 0인 정사각 행렬을 말합니다. 기호 I로 나타냅니다.

I =
1 0 0
0 1 0
0 0 1

특성

  • 단위 행렬은 숫자의 곱셈에서 1과 같은 역할을 합니다.
  • 성질: A · I = I · A = A (어떤 행렬 A에 대해서도)
  • 성질: A · A-1 = I · A-1 = A-1 · I = A-1 · A = I
  • 성질: det(I) = 1 (단위 행렬의 행렬식은 1입니다.)

머신러닝에서의 응용

행렬 연산의 기준이 되며, 역행렬을 정의하는 데 필수적입니다. 특히, 선형 회귀 모델의 정규방정식에서 역행렬을 계산할 때 단위 행렬의 특성을 활용합니다.
