머신러닝 기본 수학

머신러닝은 데이터를 수학적으로 표현하고 조작하는 과정에 깊이 의존하며, 특히 선형대수와 미적분학은 그 핵심을 이룹니다.

행렬 연산

전치 행렬 (Transpose Matrix)

기존 행렬의 행과 열을 서로 맞바꾼 행렬

예:

A = [1 2 3; 4 5 6]
AT = [1 4; 2 5; 3 6]

단위 행렬 (Identity Matrix)

주대각선의 원소는 1이고 나머지 원소는 0인 정사각 행렬

예:

I = [1 0 0; 0 1 0; 0 0 1]

역행렬 (Inverse Matrix)

정사각 행렬 A에 대해, 곱했을 때 단위 행렬이 나오는 행렬

예:

A · A-1 = I

미분과 최적화

미분의 역할

특정 지점에서 함수의 순간 변화율(기울기)을 구하는 과정

머신러닝에서 비용 함수의 최소값을 찾기 위해 필수적

경사 하강법

비용 함수 J(θ)의 기울기를 계산하여 최소값에 도달

θ = θ - α · ∂J(θ)/∂θ

α: 학습률, ∂J(θ)/∂θ: 그래디언트
