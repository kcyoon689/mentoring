주요 CNN 모델 아키텍처

CNN은 딥러닝의 발전과 함께 다양한 형태로 진화해왔으며, 여러 모델들이 이미지 분류 성능을 획기적으로 향상시켰습니다. 아래는 이미지 분류 분야에서 중요한 이정표가 된 주요 CNN 모델들의 특징과 발전 과정입니다.

AlexNet (2012)

이미지넷(ImageNet) 대회에서 처음으로 딥러닝을 적용한 모델

주요 혁신: ReLU 활성화 함수
5개의 풀링 계층
GPU로 학습 속도 개선

VGG (2014)

Visual Geometry Group에서 개발한 단순하면서 효과적인 모델

주요 혁신: 매우 작은 컨볼루션 필터(3×3)
네트워크의 깊이 증가
심층 CNN의 효과 증명

ResNet (2015)

Residual Network로, 깊은 네트워크 학습의 문제 해결

주요 혁신: Skip Connection(건너뛰기 연결)
깊은 네트워크(1000+ 층) 가능
오버피팅 문제 해결

DenseNet (2016)

Densely Connected Convolutional Network

주요 혁신: Dense Connection(밀접한 연결)
각 층이 모든 이전 층에 연결
파라미터 개수 감소

EfficientNet (2017+)

효율적인 컴퓨팅을 위한 최적화된 모델

주요 혁신: Compound Scaling
네트워크, 입력 크기, 배율의 균형
리소스 효율성 최적화

모델 크기 비교

AlexNet: 60M VGG: 130M ResNet: 25M EfficientNet: 5M~
