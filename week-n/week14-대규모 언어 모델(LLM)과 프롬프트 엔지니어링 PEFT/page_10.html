PEFT의 주요 방법론: LoRA

W = W0 + BA 형태로 업데이트

학습 시: A와 B 행렬의 파라미터만 최적화

추론 시: BA를 W0에 합쳐 원래 속도 유지

LoRA: Low-Rank Adaptation

모델의 선형 계층에 대한 업데이트를 저랭크 행렬로 근사화하는 기술

  • 기존 가중치 행렬 W0를 고정
  • 두 개의 작은 저랭크 행렬 A와 B의 곱을 추가
  • 학습 가능한 파라미터 수를 크게 줄이면서도 성능 저하 최소화

LoRA의 주요 장점

  • 파라미터 수의 극대적 감소로 GPU 메모리 사용량과 클라우드 컴퓨팅 비용 절감
  • 적은 수의 파라미터만 학습하므로 모델 업데이트 시간 단축
  • 재앙적 망각(Catastrophic Forgetting) 방지: 초기 파라미터 대부분 유지
  • 과적합 위험 감소: 전체 모델을 수정하지 않으므로 소규모 데이터셋에서도 효과적
  • 배포 용이성: 튜닝된 모델의 크기가 원본 모델보다 훨씬 작아(수십 MB 수준)
