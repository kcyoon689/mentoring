4. 핵심 PEFT 기법: LoRA 및 QLoRA

Core PEFT Techniques: LoRA & QLoRA

LoRA (Low-Rank Adaptation)
  • 기존 가중치를 고정하고 저차원 행렬(어댑터)만 학습
  • 학습 파라미터 수를 획기적으로 줄여 속도 향상
  • 성능 손실을 최소화하면서 낮은 GPU 비용으로 튜닝
QLoRA (Quantized LoRA)
  • 모델을 4비트로 양자화한 후 LoRA 기법을 적용
  • VRAM 사용량을 최대 80% 절감 가능
  • 소비자용 GPU에서도 대형 언어 모델 학습 지원
