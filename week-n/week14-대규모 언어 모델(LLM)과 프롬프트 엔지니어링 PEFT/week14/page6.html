6. 학습 방식 및 목적에 따른 분류

Classification by Learning Method & Purpose

지도 학습 파인 튜닝 (SFT)
  • 데이터셋
    레이블링된 '지시-응답' 쌍 데이터 활용
  • 주요 목적
    모델의 응답 형식 교정 및 특정 지식 체계 학습
비지도 학습 파인 튜닝
  • 데이터셋
    레이블 없는 일반 텍스트 데이터 (Raw Text)
  • 주요 목적
    의료, 법률 등 도메인 특화 지식 자체를 습득
정렬(Alignment) 튜닝
인간의 가치관과 선호도에 맞춰 모델의 답변 품질을 최적화하는 과정
RLHF: 인간 피드백 기반 보상 모델 학습 및 강화학습 수행
DPO: 보상 모델 없이 선호도 데이터를 직접 최적화 (효율적)
* DPO는 RLHF 대비 선호 점수 150% 향상 사례 존재
