통계적 접근과 딥러닝의 등장

통계적 접근 (1990-2000s)

NLP 연구가 규칙 기반에서 통계적 모델로 전환

대규모 텍스트 코퍼스를 활용한 언어 패턴 분석

주요 모델

  • N-gram 모델: 단어 시퀀스의 확률을 예측
  • 은닉 마르코프 모델(HMM): 품사 태깅, 개체명 인식, 기계 번역

한계

당시의 제한적인 컴퓨팅 파워로 인해 대규모 프로젝트 진행 어려움

딥러닝의 등장 (2010s)

딥러닝과 신경망 아키텍처의 발전이 NLP 분야에 혁명을 가져옴

순환 신경망(RNN)은 순차적인 데이터 처리에 특화된 모델

RNN은 문맥을 기억하며 언어를 처리할 수 있음

주요 모델

RNN

순차적 데이터 처리의 기본 모델

LSTM

장단기 메모리 모델로 기울기 소실 문제 완화

GRU (Gated Recurrent Unit)

LSTM의 간소화된 변형으로 더 적은 매개변수로 효율적인 시퀀스 처리

RNN의 한계

긴 문맥을 처리할 때 '기울기 소실(gradient vanishing)' 문제에 직면하여 장기 의존성을 포착하는 데 한계가 있음

텍스트 데이터와 임베딩: 자연어 처리의 발전 과정
