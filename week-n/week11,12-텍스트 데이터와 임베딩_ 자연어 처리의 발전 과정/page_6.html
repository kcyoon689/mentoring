예측 기반 임베딩: Word2Vec과 GloVe

원-핫 인코딩과 LSA의 한계를 보완하며 등장한 예측 기반 임베딩 기법들은 단어의 의미적 관계를 효과적으로 수치로 표현합니다.

Word2Vec

토히마스 미콜로프, 2013

학습 방식:

CBOW: 주변 단어로 중간 단어 예측

Skip-gram: 중간 단어로 주변 단어 예측

특징:

단어 간 의미 유사성 잘 포착

전체 코퍼스 정보 미반영

OOV 문제 처리 어려움

GloVe

스坦포드 대학, 2014

학습 방식:

동시 등장 행렬의 통계 정보 활용

내적을 통해 동시 등장 확률 로그값 근사

특징:

전체 코퍼스 통계 반영

단어 유추 성능 우수

계산 복잡성 (대규모 행렬 처리)

FastText

Word2Vec의 확장

학습 방식:

단어를 쪼개서도 임베딩

하나의 단어에 여러 하위 단어(subword) 존재

특징:

오타나 합성어 처리 향상

OOV 문제 개선 (하위 단어 활용)

예: 'apple' → 'ap', 'app', 'ppl', 'ple', 'le'

Word2Vec vs GloVe 비교

기반
Word2Vec: 예측 기반 (신경망)
GloVe: 카운트 기반 + 예측 기반 (하이브리드)
문맥 처리
윈도우 크기 내의 지역적 문맥
전체 코퍼스의 전역적 동시 등장 정보
장점
단어 간 의미 유사성 포착 우수
전체 코퍼스 통계 반영, 단어 유추 성능 우수
단점
전체 코퍼스 정보 미반영
계산 복잡성 (대규모 행렬 처리)
텍스트 데이터와 임베딩: 자연어 처리의 발전 과정
6/10
