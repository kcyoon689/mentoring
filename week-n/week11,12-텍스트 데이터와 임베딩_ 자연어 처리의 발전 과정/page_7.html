문맥을 고려한 동적 임베딩: ELMo

다의어(polysemy) 문제

자연어에는 '은행(bank)'처럼 문맥에 따라 의미가 달라지는 단어가 많습니다. 고정된 벡터 표현은 이러한 문맥 의존적 의미를 표현할 수 없습니다.

ELMo의 접근 방식

  • 양방향 LSTM(Long Short-Term Memory)을 기반으로 하는 언어 모델
  • 단어 자체의 의미뿐만 아니라 문장 내에서 해당 단어가 사용된 문맥을 고려
  • 양방향 LSTM을 통해 문장의 앞뒤 정보를 모두 학습
  • 각 계층의 출력을 결합하여 최종 임베딩 생성

양방향 LSTM 동작 방식

ELMo의 주요 장점

동적 임베딩

고정된 단어 임베딩이 아닌, 문맥에 따라 동적으로 변화하는 임베딩 제공

문맥 이해

단어의 문맥적 의미를 정확히 파악하여 더 풍부한 표현 가능

성능 향상

NLP 모델의 성능을 크게 향상시키며, 특히 복잡한 언어 이해 작업에서 효과적
