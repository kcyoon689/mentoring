Transformer의 한계: O(n²) 연산 비용 문제

Attention 메커니즘의 복잡도 문제

  • Transformer의 핵심인 Attention 메커니즘은 입력 시퀀스 길이에 대해 O(n²)의 연산 비용을 가집니다
  • 입력 토큰의 길이가 길어질수록 토큰 쌍마다 유사도를 계算해야 하므로, 연산량과 메모리 사용량이 제곱으로 증가합니다
  • 컨텍스트 윈도우를 확장할수록 비용이 폭발적으로 증가하며, 긴 문서를 처리하는 데 구조적 한계가 발생합니다

결과적 영향

  • 긴 문서 처리 시 성능이 급격히 저하
  • GPU 메모리 소비가 커서 대규모 모델 실행 어려움
  • 실제 응용 프로그램에서 긴 컨텍스트 활용 제한
