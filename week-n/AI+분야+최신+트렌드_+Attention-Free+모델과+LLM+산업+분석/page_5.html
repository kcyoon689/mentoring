Attention-Free 모델의 등장

Transformer의 한계를 극복하기 위해 Attention 메커니즘을 제거하거나 그 비중을 줄이는 새로운 아키텍처들이 활발히 연구되고 있습니다.

RWKV

  • 전통적인 RNN과 유사한 구조
  • 시간 의존성 문제 해결

Mamba

  • State Space Model(SSM) 기반
  • 선형적 확장성(O(n))을 가짐
  • 최대 5배 빠른 추론 속도

RetNet

  • 이전 정보를 계속 업데이트하는 순환 구조
  • Transformer의 Attention 레이어 대체
  • 모든 단어를 서로 비교하지 않음

Power Retention

  • Brumby-14B 모델 포함
  • Transformer의 어텐션 레이어를 완전히 제거
  • 긴 컨텍스트 처리에 특화

computational complexity

  • Transformer: O(n²) - 토큰 쌍마다 유사도 계산
  • Attention-free: O(n) - 선형적 확장성
