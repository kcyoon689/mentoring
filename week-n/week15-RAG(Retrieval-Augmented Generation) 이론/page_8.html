고급 RAG 기법

기본적인 RAG는 검색된 문서를 단순히 LLM에 전달하는 방식이지만, 고급 RAG 기법들은 검색 및 생성 과정을 더욱 정교하게 제어하고 개선하여 성능을 극대화합니다.

Self-Corrective RAG (자가 교정 RAG)

이 기법은 LLM이 생성한 답변의 품질을 스스로 평가하고, 필요한 경우 검색 쿼리를 수정하거나 추가 정보를 검색하여 답변을 개선하는 방식입니다.

최종 답변 생성

LLM의 추론 능력 활용

LLM의 추론 능력을 활용하여 검색 과정을 제어하고, 답변의 품질을 스스로 평가합니다.

정확성 향상

초기 답변의 오류나 불충분한 정보를 감지하면, 내부적으로 새로운 검색 전략을 수립하고 재검색을 수행하여 최종 답변의 정확성을 높입니다.

특징: Self-Corrective RAG은 LLM의 자기 평가 및 자기 개선 능력을 활용하여, 검색 및 생성 과정을 반복적으로 최적화합니다. 이는 특히 복잡한 질문이나 도메인 특화된 쿼리에서 유용합니다.
