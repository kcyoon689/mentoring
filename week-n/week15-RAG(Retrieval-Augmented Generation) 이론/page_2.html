RAG 소개

RAG(Retrieval-Augmented Generation)은 대규모 언어 모델(LLM)의 한계를 극복하기 위해 등장한 혁신적인 기술입니다.

LLM은 방대한 데이터를 학습하여 뛰어난 언어 생성 능력을 보여주지만, 여러 문제점을 가지고 있습니다. RAG는 외부 지식 소스에서 관련 정보를 검색하여 LLM의 답변 생성 과정을 보강함으로써 이러한 문제들을 해결합니다.

LLM의 한계점

최신 정보 부족

학습 데이터 이후로 생성된 정보에 접근할 수 없습니다.

환각(Hallucination) 현상

사실이 아닌 정보를 생성하거나, 존재하지 않는 사실에 대해 진술합니다.

특정 도메인 지식 부재

특정 분야의 전문 용어나 지식에 대해 제한적으로 이해하거나 답변할 수 없습니다.

RAG의 해결 방안

검색기(Retriever)와 생성기(Generator)의 두 단계로 구성된 RAG는 외부 지식 소스에서 관련 정보를 검색하여 LLM의 답변 생성 과정을 보강합니다.
