객체 인식의 주요 접근 방식

머신러닝 기반 접근 방식

특징 추출

수동 특징 추출 (예: HOG, SURF, MSER)

분류

SVM, Bag-of-Words 모델, Viola-Jones 알고리즘 등

데이터 요구량

최소한의 데이터로도 정확한 결과 가능

훈련 방식

특징 추출 알고리즘으로 데이터에서 특징을 선택 후 머신러닝 모델에 추가하여 분류

장점

다양한 특징 및 분류기 조합 가능, 적은 데이터로도 효과적

단점

특징 추출 과정의 수동성, 복잡한 패턴 인식의 한계

딥러닝 기반 접근 방식

특징 추출

자동 특징 학습 (예: CNN을 통한 특징 자동 학습)

분류

CNN 기반 모델 (예: AlexNet, GoogLeNet)

데이터 요구량

높은 수준의 정밀도를 위해 대량의 레이블링된 데이터 필요

훈련 방식

CNN과 같은 딥러닝 모델이 객체 고유의 특징을 자동으로 학습

장점

높은 정밀도 제공, 복잡한 특징 자동 학습

단점

대량의 데이터와 고성능 GPU 필요
