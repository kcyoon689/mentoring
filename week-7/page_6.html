이미지 세그멘테이션(Image Segmentation)의 정의 및 목표

정의

이미지 세그멘테이션은 컴퓨터 비전 기술의 한 분야로, 이미지 내의 각 픽셀을 특정 객체 또는 영역에 따라 분류하는 것을 목표로 합니다.

이는 단순히 객체의 존재 여부나 위치를 파악하는 객체 인식(Object Recognition)을 넘어, 이미지의 모든 픽셀에 대해 의미론적 레이블을 할당함으로써 객체의 정확한 형태와 경계를 이해하는 데 중점을 둡니다.

다른 컴퓨터 비전 작업과의 비교

이미지 분류

이미지 전체를 하나의 클래스로

객체 탐지

객체의 존재와 위치 파악

이미지 세그멘테이션

모든 픽셀의 클래스 할당

이러한 픽셀 단위의 분석을 통해 컴퓨터는 사람처럼 이미지를 구성하는 요소들을 깊이 있게 이해하고 해석할 수 있게 됩니다.

주요 목표

각 픽셀의 의미론적 레이블 할당

이미지의 모든 픽셀이 어떤 의미론적 범주(예: 도로, 하늘, 사람, 자동차)에 속하는지 분류

객체의 정확한 형태와 경계 파악

객체 인식보다 훨씬 더 세밀하고 정밀한 분석 제공, 객체의 경계까지 정확히 파악

컴퓨터의 인간식 시각 이해

픽셀 단위 분석을 통해 컴퓨터가 사람처럼 이미지를 구성하는 요소들을 깊이 있게 이해하고 해석

픽셀 수준의 분석

이미지 세그멘테이션은 픽셀 단위로 이미지를 분석하여 각 픽셀이 어떤 객체에 속하는지 결정합니다.

이렇게 함으로써 객체의 정확한 위치, 크기, 모양, 그리고 객체 간의 관계를 파악할 수 있습니다.
