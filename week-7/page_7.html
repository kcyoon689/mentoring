이미지 세그멘테이션의 유형

이미지 세그멘테이션은 분석의 정밀도와 목표에 따라 크게 세 가지 유형으로 나뉩니다.

시맨틱 세그멘테이션

정의: 이미지 내의 모든 픽셀에 대해 해당 픽셀이 속하는 클래스 레이블을 할당합니다.

목표: 이미지의 각 픽셀이 어떤 의미론적 범주에 속하는지 분류합니다.

결과물: 동일한 클래스에 속하는 모든 객체는 하나의 동일한 레이블로 표시됩니다.

예시: 거리 이미지에서 모든 나무를 '나무' 클래스로, 모든 차량을 '차량' 클래스로 주석 처리합니다.

Semantic Segmentation

인스턴스 세그멘테이션

정의: 시맨틱 세그멘테이션과 유사하게 픽셀에 클래스 레이블을 할당하지만, 동일한 클래스 내의 개별 객체를 서로 구분합니다.

목표: 각 픽셀이 어떤 클래스에 속하는지, 해당 클래스 내의 어떤 개별 객체에 속하는지까지 식별합니다.

결과물: 동일한 클래스에 속하더라도 서로 다른 객체는 고유한 레이블 또는 ID를 가집니다.

예시: 거리 이미지에 있는 수많은 자동차를 각각 개별적인 인스턴스로 인식하고 분할합니다.

Instance Segmentation

파놉틱 세그멘테이션

정의: 시맨틱 세그멘테이션과 인스턴스 세그멘테이션의 장점을 결합한 유형입니다.

목표: 이미지의 모든 픽셀에 대해 클래스 레이블을 할당하면서, 동시에 개별 객체(Thing 클래스)는 인스턴스별로 구분하고, 배경(Stuff 클래스)은 시맨틱하게 분할합니다.

결과물: 이미지 전체에 대한 포괄적인 장면 이해를 제공하며, 모든 픽셀이 분류되고 개별 객체는 구분됩니다.

예시: 도로, 보도, 나무와 같은 배경 영역은 시맨틱하게 분할하고, 자동차, 보행자와 같은 개별 객체는 인스턴스별로 분할합니다.
