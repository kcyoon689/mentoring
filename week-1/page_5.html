상속과 다형성

상속과 다형성은 코드 재사용성을 높이고 유연한 설계를 가능하게 하는 OOP의 중요한 기둥입니다.

상속 (Inheritance)

상속은 기존 클래스(부모 클래스 또는 슈퍼클래스)의 속성과 메서드를 물려받아 새로운 클래스(자식 클래스 또는 서브클래스)를 만드는 기능입니다. 이를 통해 코드 중복을 줄이고 클래스 간의 계층적 관계를 형성할 수 있습니다.

class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        raise NotImplementedError("Subclass must implement abstract method")

class Dog(Animal):
    def speak(self):
        return f"{self.name} barks"

class Cat(Animal):
    def speak(self):
        return f"{self.name} meows"
Animal (부모 클래스)
Dog (자식 클래스)
Cat (자식 클래스)

자식 클래스는 부모 클래스의 기능을 그대로 사용하거나, 필요에 따라 기능을 추가하거나 변경(오버라이딩)할 수 있습니다.

다형성 (Polymorphism)

다형성은 '여러 형태를 가질 수 있는 능력'을 의미하며, 서로 다른 클래스의 객체가 동일한 메서드 호출에 대해 각자의 방식대로 응답하는 것을 말합니다. 이는 주로 메서드 오버라이딩을 통해 구현됩니다.

animals = [Dog("Buddy"), Cat("Whiskers")]

for animal in animals:
    # 같은 speak() 메서드를 호출하지만, 객체의 실제 타입에 따라 다른 결과가 출력됨
    print(animal.speak())
speak()
Buddy barks
Whiskers meows

객체의 구체적인 타입에 의존하지 않고, 공통된 인터페이스를 사용하여 코드를 작성할 수 있어 프로그램이 더 유연해집니다.

* 코드 예제는 Python 3.x 기준입니다.
