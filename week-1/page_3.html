모듈과 패키지 관리

외부 라이브러리를 사용하기 위해서는 패키지 관리 도구가 필요합니다. 파이썬의 표준 패키지 관리자는 pip로, 간단한 명령어를 통해 라이브러리를 설치, 업데이트, 삭제할 수 있습니다.

패키지 설치

  • 패키지 설치 명령어

    원하는 라이브러리를 설치할 때 사용하는 기본 명령어입니다.

  • 업데이트 명령어

    이미 설치된 패키지를 최신 버전으로 업데이트할 때 사용합니다.

  • 삭제 명령어

    필요하지 않은 패키지를 삭제할 때 사용합니다.

패키지 설치:
$ pip install <package_name>
패키지 업데이트:
$ pip install --upgrade <package_name>
패키지 삭제:
$ pip uninstall <package_name>

가상 환경

  • 가상 환경의 목적

    프로젝트마다 독립된 실행 환경을 만들어 패키지 간의 의존성 충돌을 방지하는 것입니다.

  • 가상 환경 도구

    파이썬 3.3 이상에서는 venv
