모듈과 패키지 관리

파이썬의 표준 패키지 관리자는 pip이며, 가상환경 관리에는 Conda나 uv를 사용할 수 있습니다.

패키지 설치 (pip)

  • 설치

    PyPI에서 원하는 라이브러리를 설치합니다.

  • 업데이트

    설치된 패키지를 최신 버전으로 갱신합니다.

  • 삭제

    불필요한 패키지를 제거합니다.

설치:
$ pip install <package_name>
업데이트:
$ pip install --upgrade <package_name>
삭제:
$ pip uninstall <package_name>

가상 환경 & 패키지 (uv)

  • 특징

    초고속 패키지/환경 관리. pip/venv 호환 가능.

  • 사용법

    pip 호환: uv pip ...
    프로젝트 지향: uv add/remove

venv 생성:
$ uv venv .venv
활성화:
$ source .venv/bin/activate
패키지 설치/삭제:
$ uv pip install <package>
$ uv pip uninstall <package>
프로젝트 지향:
$ uv add numpy
$ uv remove numpy

가상 환경 (Conda)

  • 목적

    프로젝트별 독립 환경과 라이브러리 충돌 방지.

  • 특징

    Python 외 과학계산·시스템 라이브러리까지 관리.

환경 생성:
$ conda create -n <env_name> python=3.10
활성화/비활성화:
$ conda activate <env_name>
$ conda deactivate
패키지 설치:
$ conda install numpy
환경 목록/삭제:
$ conda env list
$ conda remove -n <env_name> --all
