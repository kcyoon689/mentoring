기초 통계 분석

데이터 사이언스의 기초는 통계학입니다. 기술 통계와 추론 통계는 데이터를 이해하고, 모델을 만들며, 인사이트를 도출하는 데 필수적인 역할을 합니다.

기술 통계

중심 경향치

평균 (Mean)

산술 평균

중앙값 (Median)

데이터의 중앙값

최빈값 (Mode)

가장 빈번한 값

import statistics, numpy as np, pandas as pd

x = [8.0, 1, 2.5, 4, 28.0]

# 평균 계산
mean_stat = statistics.mean(x)  # 8.7
mean_np = np.mean(x)            # 8.7

# 중앙값 계산
median_stat = statistics.median(x) # 4.0
median_np = np.median(x)           # 4.0

분산도

분산 (Variance)

편차 제곱의 평균

표준편차 (Standard Deviation)

분산의 제곱근

import statistics, numpy as np

x = [8.0, 1, 2.5, 4, 28.0]

# 분산 계산
var_stat = statistics.variance(x)      # 110.2
var_np = np.var(x, ddof=1)             # 110.2

# 표준편차 계산
std_stat = statistics.stdev(x)         # 10.4976...
std_np = np.std(x, ddof=1)             # 10.4976...

추론 통계

추론 통계는 수집된 표본 데이터를 바탕으로 더 큰 모집단의 특성을 추론하고 예측하는 방법입니다.

가설 검정 (Hypothesis Testing)

모집단에 대한 가설을 설정하고, 표본 데이터를 통해 그 가설이 맞는지 통계적으로 검증하는 과정입니다.

from statsmodels import api as sm

# F-검정 예
result = sm.OLS(y, X).fit()
f_statistic = result.fvalue
p_value = result.f_pvalue

회귀 분석 (Regression Analysis)

하나 이상의 독립 변수가 종속 변수에 미치는 영향을 파악하고, 이를 통해 미래 값을 예측하는 모델을 만드는 통계 기법입니다.

import statsmodels.api as sm

# 선형 회귀 모델
model = sm.OLS(y, X)
results = model.fit()
predictions = results.predict(X_new)
